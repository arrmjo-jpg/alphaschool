# BUS-0012: Competency / Skill Framework — Proposed, Not Yet Decided

## Context

During a "find genuinely missing entities" discovery pass (explicitly forgetting existing LMS products, thinking from first principles), a Competency/Skill Framework was identified as a Critical-priority gap: an independent taxonomy of skills/competencies, referenced by courses/assessments/badges, queried across courses rather than owned by any single one.

## Problem

Should AlphaSchool build a Competency Framework, and if so, in what phase?

## Future Implications

If accepted: this is the mechanism "Teacher Training" and "Staff Training" need to mean something more than course completion — answering questions like "which teachers are qualified to teach X" or tracking expiring professional certifications. Without it, those two named platform requirements reduce to plain completion tracking.
